## The Problem

Building an AI companion app solo means every layer — product design, the conversational experience, and the infrastructure behind it — has to be handled by one person, end to end.

## What I Built

CatChat is a Next.js web app, built solo in TypeScript, using Firebase for auth and data and the OpenAI API for the AI companion's conversational core. I owned the full product lifecycle — design, engineering, and deployment.

## Key Challenges

As a solo build, the constraint was scope discipline: making deliberate calls about what the companion experience needed versus what could be cut to ship. On the technical side, shaping the OpenAI API prompts and conversation state management in Firebase to give the companion a consistent, coherent personality across sessions was the core problem.

## Results

CatChat shipped as a complete solo product — from initial design through deployment — with full ownership across the stack.

## Stack Decisions

Next.js and Firebase were chosen together as a fast path to a production web app without standing up custom backend infrastructure, which mattered given this was a solo, self-funded build.
