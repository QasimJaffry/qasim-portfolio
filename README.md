# Qasim Hassan — Portfolio

Personal portfolio site with case studies for selected mobile, web, and AI products.

**Live:** [qasimjaffry.github.io/qasim-portfolio](https://qasimjaffry.github.io/qasim-portfolio/)

## What’s here

- Featured work (Agenticly, Innerverse, Kitty Nip, BugMapper, Safe Deal, Decidr, and more)
- Project case studies (MDX) with stack, role, and outcomes
- About / experience / contact

Built as a static export and deployed to **GitHub Pages**.

## Stack

- Next.js 15 (App Router) · React 19 · TypeScript · Tailwind CSS 4
- MDX case studies (`next-mdx-remote` + `gray-matter`)
- Static export (`output: "export"`) with `basePath` for GitHub Pages

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

```bash
npm run build   # static export → out/
npm run lint
```

## Deploy

Pushes to `main` trigger GitHub Actions (`.github/workflows/deploy.yml`) and publish to GitHub Pages.
